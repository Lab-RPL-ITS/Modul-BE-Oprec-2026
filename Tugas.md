# Tugas Materi Role Backend - Oprec Admin RPL 2026.

## Deskripsi Tugas

Mas Aldeen merasa bahwa buku-buku di Lab RPL sangat bermanfaat dan ingin agar lebih banyak mahasiswa dapat mengakses serta membacanya secara luas. Dari situ, ia memiliki ide untuk membuat sebuah sistem library management yang dapat digunakan untuk membagikan koleksi buku-buku tersebut, sehingga mahasiswa lain bisa dengan mudah meminjamnya. Namun, karena Mas Aldeen sibuk banget, ia menugaskan para calon admin baru untuk mengembangkan sebuah API yang nantinya akan digunakan oleh aplikasi ini, yang direncanakan akan diberi nama **RPLibrary**.

Berdasarkan kebutuhan dan tujuan yang ingin dicapai dari aplikasi tersebut, maka disusun beberapa user story berikut:

1. Sebagai calon pengguna, saya ingin mendaftar ke dalam sistem menggunakan data diri saya, sehingga saya bisa memiliki akun untuk meminjam buku.
2. Sebagai pengguna terdaftar, saya ingin login menggunakan email dan kredensial saya, sehingga saya bisa mengakses fitur-fitur perpustakaan.
3. Sebagai admin, saya ingin fitur tertentu dibatasi berdasarkan role, sehingga data dan keamanan sistem tetap terjaga.
4. Sebagai admin, saya ingin menambah, melihat, mengubah, dan menghapus (CRUD) data buku, sehingga katalog perpustakaan selalu up-to-date.
5. Sebagai Admin, saya ingin mengunggah file gambar saat menambahkan buku baru, sehingga buku tersebut memiliki cover/sampul yang bisa dilihat oleh pengunjung.
6. Sebagai Pengguna, saya ingin mengunggah file gambar untuk data akun, sehingga saya memiliki foto profil untuk akun saya.
7. Sebagai pengguna, saya ingin meminjam buku yang tersedia melalui sistem, sehingga saya bisa membawa dan membacanya.
8. Sebagai admin, saya ingin memproses dan mengonfirmasi pengembalian buku di sistem setelah pengguna menyerahkan fisik bukunya di Lab, sehingga status buku kembali tersedia.

Optional
1. Sebagai admin, saya ingin mengelola (CRUD) tags atau kategori buku, sehingga saya bisa mengelompokkan buku untuk memudahkan pencarian.
2. Sebagai pengguna, saya ingin mencari dan memfilter daftar buku di katalog, sehingga saya bisa dengan cepat menemukan buku yang ingin saya baca.
